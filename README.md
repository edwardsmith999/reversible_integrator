# reversible_integrator
A Fortran 90 implementation of "Molecular Dynamics and Time Reversibility" by D. Levesque I and L. Verlet Journal of Statistical Physics, Vol. 72, Nos. 3/4, 1993 with additional details from:

Bit-Reversible Version of Milne’s Fourth-Order Time-Reversible Integrator for Molecular Dynamics Wm.G. Hoover, C.G. Hoover CMST 23(4) 299–303 (2017) DOI:10.12921/cmst.2017.0000031

and

Time-Irreversibility is Hidden Within Newtonian Mechanics by William Hoover and Carol Hoover (2018)
